Release Note 1.1.0 (2025 Dec - 2026 Jan)

New Features & Improvements

**Blockchain & Wallet Integration**
- Metamask Wallet: Now available over Solana network, expanding multi-chain support.

- Improve queue position correction in the production environment.

**Leaderboard Management**
